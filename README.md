# LeadFlow — Frontend

> React 18 · Vite · TypeScript · Tailwind · Redux Toolkit · React Router v6

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build        # stage build
npm run build:prod   # production build
```

## Environment Setup

Copy and fill in `environments/.env.local`:

```bash
VITE_API_URL=http://localhost:3001
VITE_APP_URL=http://localhost:5173
```

## Testing

```bash
npm test
npm run test:ui
```
